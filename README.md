# Netflix Clone – Movie Streaming Platform  

A dynamic web application built using **React**, featuring user authentication, protected routes, and API-driven movie content for an engaging browsing experience.  

### Features  
- **User Authentication**: Implemented secure signup and login functionality with user data stored in the database and protected route access.  
- **Dynamic Movie Browsing**: Fetched and displayed categorized movie data using external APIs, enabling users to explore content based on their preferences.  
- **Personalized Experience**: Provided a custom homepage after login with seamless navigation and logout functionality.
